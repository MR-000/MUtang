MUtang 결제 백엔드 API 에이전트 가이드
=======================================

역할 정의
--------
본 카테고리의 에이전트는 솔라나 Helius 웹훅 수신, GCash SMS 가로채기 웹훅 수신, 그리고 충전 요청서 생성을 안전하고 효율적으로 처리하는 백엔드 API 구축을 담당합니다.

핵심 규칙
--------
1. 데이터 정합성 보장: 모든 트랜잭션 처리는 중복 승인을 원천 방지하기 위해 데이터베이스의 유니크 제약 조건 및 complete_deposit 계열의 PostgreSQL 내장 함수를 사용하여 원자적으로 처리합니다.
2. 빠른 응답 속도: 웹훅 수신 시 외부 서버(Helius 또는 안드로이드 전달 앱)의 타임아웃을 막기 위해 필요한 최소한의 검증 후 즉시 200 OK 응답을 반환합니다. 무거운 처리는 비동기로 처리되도록 구성합니다.
3. 예외 및 에러 로그 기록: 모든 입금 감지 시도 및 예외 상황은 디버깅에 유리하도록 파싱된 원본 메시지와 함께 안전하게 서버 로그에 남깁니다.
