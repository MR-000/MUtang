MUtang 충전 UI 에이전트 가이드
==============================

역할 정의
--------
본 카테고리의 에이전트는 사용자가 모바일 환경에서 GCash 및 솔라나 달러 토큰(USDT/USDC)으로 크레딧을 신청하고, 직관적인 3단계 체크박스 충전 단계를 통해 실시간 입금 확인 과정을 지켜볼 수 있는 사용자 친화 충전 페이지(deposit page) 구축을 담당합니다.

핵심 규칙
--------
1. 모바일 우선 및 고대비 최적화: 필리핀 현지의 저사양 기기 환경에서도 레이아웃 틀어짐 없이 빠르게 로딩되며 가독성이 매우 뛰어난 고대비 Tailwind UI를 구현합니다.
2. 직관적인 충전 단계 시각화: 3단계 진행 체크박스 및 실시간 3분 만료 타이머를 아주 쉽게 노출하여 비전공자 소상공인도 자신의 결제가 어떤 상태에 있는지 한눈에 알 수 있게 돕습니다.
3. 실시간 동기화: 백엔드에서 입금이 감지되는 즉시 Supabase Realtime 또는 효율적인 Polling 방식을 사용하여 화면의 진행 단계를 자동으로 성공 상태로 갱신시킵니다.
