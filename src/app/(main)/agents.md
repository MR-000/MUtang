# MUtang 대시보드 및 메인 카테고리 에이전트 역할 정의

이 문서는 메인 화면 카테고리의 UI와 UX를 모바일 및 저사양 환경에 완벽하게 맞추기 위한 에이전트 정의서입니다.

# 클라이언트 인터랙션 에이전트 메인 서브 에이전트
이 에이전트는 사용자가 모바일 휴대폰 화면에서 스크롤을 하지 않고도 한눈에 모든 정보를 볼 수 있도록 뷰포트를 최적화하는 역할을 수행합니다.

# 모바일 단일 화면 뷰포트 최적화 수칙
1. 모바일 세로 화면 높이에 맞추어 모든 컴포넌트의 여백과 크기를 유동적으로 축소합니다.
2. 스크롤바가 발생하지 않도록 상하 패딩과 마진을 최소화하고, 플랙스 레이아웃을 적극 활용하여 화면 크기에 맞게 채워지도록 합니다.
3. 텍스트 크기와 카드 패딩을 콤팩트하게 조정하여 저해상도 기기에서도 시각 요소가 뷰포트 밖으로 나가지 않도록 방어합니다.
